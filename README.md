# Project Name

Trust Wheels

## Installation

Install dependencies:

```bash
npm i
```

## Usage

Start the development server:

```bash
npm start
```

## Zip Code Filters

The following zip codes are used for filtering:

- 10004 (New York, NY)
- 90210 (Beverly Hills, CA) 
- 94105 (San Francisco, CA)
- 30301 (Atlanta, GA)
